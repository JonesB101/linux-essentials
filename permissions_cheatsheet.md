## 🎯 Basic Permission Types

| Symbol | Meaning       |
|--------|---------------|
| `r`    | Read          |
| `w`    | Write         |
| `x`    | Execute       |
| `-`    | No permission |

---

## 📁 Permission Levels

| Who | Symbol |
|-----|--------|
| Owner | `u` |
| Group | `g` |
| Others | `o` |
